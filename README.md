# BaiCiZhanPrinter

将手机端APP 百词斩中的已背单词表导出。

## 使用方法


1. 下载本项目并安装依赖
   1. 本项目依赖 `python3` 和 `xlwt`，请确保已安装
2. 手机端导出数据文件
   1. 手机端进入`Android/data/com.jiongji.andriod.card/files/baicizhan`
   2. 导出`baicizhantopicproblem.db`（包含已背的单词）和`lookup.db`（包含所有单词）到电脑
3. 运行`main.py`
